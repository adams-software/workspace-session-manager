# msr

Minimal Session Runtime (core library)

This repository is the first effort toward a small, explicit session runtime core
for a future richer application stack.

## Scope (current)
- library-first runtime primitives
- single-session lifecycle + attach/detach semantics
- explicit API contract in `docs/specs/minimal-session-runtime.md`

## Non-goals (current)
- manager UX/navigation commands
- logging/replay
- TUI
